# NoteStack

Modern ve kullanıcı dostu Python masaüstü not alma uygulaması.

## Özellikler

- Modern ve şık arayüz (CustomTkinter)
- JSON ile veri saklama
- Hızlı ve performanslı
- Özelleştirilebilir
- Not validasyonu

## Kurulum

```bash
pip install -r requirements.txt
python main.py
```

## Teknolojiler

- Python 3.10+
- CustomTkinter (Modern UI)
- JSON (Veri saklama)
